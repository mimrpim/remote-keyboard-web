# Remote Keyboard Web

An easy keyboard for using your iPad (Or other device) as your wireless keyboaed (It can be used for gaming, tesed on Minecraft Singleplayer)

## How to install

1. Install python 3.12.10 or later (Later versions not tested, there is no support for other versions than 3.12.10)
2. Run dependings.bat or dependings.sh (depenging what OS you are using)
3. Run run_services.py
4. Open the web on your device, where you want to use the keyboard
5. (You can skip tihs step if you want) Add this web to the desktop (on iOS it is by clicking share and choosing add Desktop Shortcut)
6. Enter the IP, that you used for connecitng to the web server and add :808 to it
7. Test if, if it works (For bugs or features use Issues or mail me on [mimrpim@gmail.com](mailto:mimrpim@gmail.com)
